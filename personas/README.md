# Market Hub — User Personas

Profiles of the AgriSafe executives and analysts who interact with the Market Hub platform. Each persona maps to specific modules, routines, and automation opportunities.

Source profiles: `agents/stakeholders/agrisafe/` (full organizational personas with persona.md and skills.md).

## Automation Tiers

| Tier | Description | Examples |
|------|-------------|---------|
| **Automated** | Runs daily via cron, no human input | Market data sync, news ingestion, judicial recovery sync |
| **AI-Assisted** | Virtual coworker generates draft, human reviews | Content ideas, news summaries, competitor signal analysis |
| **Manual** | Requires human judgment or external data | Campaign planning, CRM updates, event scheduling, retailer imports |
